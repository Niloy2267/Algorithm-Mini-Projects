# Algorithm Mini Projects in C++

A collection of classical algorithm implementations in C++ developed as mini projects for Algorithm Design and Analysis.

## Projects Included

### 1. Fractional Knapsack
- Algorithm Paradigm: Greedy Algorithm
- Objective: Maximize profit within a fixed knapsack capacity using divisible items.
- Time Complexity: O(n log n)

### 2. Dijkstra's Shortest Path Algorithm
- Algorithm Paradigm: Graph Algorithm
- Objective: Find the shortest path between cities in a weighted graph.
- Time Complexity: O((V + E) log V)

### 3. Towers of Hanoi
- Algorithm Paradigm: Recursion
- Objective: Solve the classic disk-moving puzzle recursively.
- Time Complexity: O(2^n)
